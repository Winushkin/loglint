# loglint

`loglint` - линтер, который проверяет текст сообщений в логах `log/slog` и `go.uber.org/zap`.

## Что проверяет

- `checkLowerStart` - сообщение должно начинаться с маленькой буквы.
- `checkEnglish` - в сообщении разрешены только английские буквы.
- `checkSpecialChars` - запрет спец символов.
- `checkSensitiveData` - список чувствительных слов (без учета регистра).

## Установка

1. Подключите плагин:
```bash
  make start
```
После этого появится файл `golangci`

## Использование

1. Перенесите файл `golangci` в директорию нужного проекта:
```bash
  cp ./golangci <relative/path/to/your/project>
```

2. Запускайте линтер через полученный исполняемый файл:
```bash
  ./golangci run ./<package/to/lint>/...
```

## Тестирование
Можно запустить тестирование линтера:
```bash
  go test ./internal/... -v
```
